# MomFlow Wilailak 📱

แอปเดียวจบสำหรับแม่เลี้ยงเดี่ยว — จัดการงาน นัดหมาย เงิน และคุยกับ "จุ้ย" ผู้ช่วย AI ส่วนตัว

## วิธีใช้งาน (เปิดทันที ไม่ต้องติดตั้งอะไร)

1. แตกไฟล์ ZIP นี้ออกมาทั้งหมดในโฟลเดอร์เดียวกัน
2. ดับเบิลคลิกเปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ (Chrome แนะนำสุด)
3. ใช้งานได้ทันที — ข้อมูลทั้งหมดจะถูกเก็บไว้ในเครื่อง/เบราว์เซอร์นั้น (localStorage)

## วิธีติดตั้งเป็นแอปบนมือถือ (เพิ่มหน้าจอโฮม)

วิธีที่ดีที่สุดคือการอัปโหลดขึ้นเว็บโฮสติ้งฟรีก่อน (เช่น GitHub Pages) แล้วเปิดผ่าน Chrome บนมือถือ:

1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น GitHub repository (เช่นเดียวกับเว็บไซต์ juybakerry ที่เคยทำไว้)
2. เปิด Settings > Pages > เลือก branch main แล้วกด Save
3. รอ 1-2 นาที จะได้ลิงก์ เช่น `https://ชื่อบัญชี.github.io/ชื่อ-repo/`
4. เปิดลิงก์ด้วย Chrome บนมือถือ
5. แตะปุ่มเมนู (จุดสามจุด) > "เพิ่มไปยังหน้าจอโฮม" หรือ "ติดตั้งแอป"
6. ไอคอนแอป MomFlow จะไปอยู่ที่หน้าโฮมเหมือนแอปทั่วไป เปิดใช้แบบออฟไลน์ได้ทันที

## ถ้ากดเปิดลิงก์แล้วไม่มีตัวเลือก "ติดตั้งแอป" ขึ้นมา

ตอนนี้แอปมีปุ่ม "ติดตั้งแอป" อยู่ในหน้า ตั้งค่า (ไอคอนเฟืองมุมขวาบน) แล้ว แต่ถ้ายังไม่ขึ้น ให้ลองตามนี้:

1. **ต้องเปิดผ่านลิงก์ https:// เท่านั้น** (เช่นลิงก์ GitHub Pages) — ถ้าเปิดไฟล์ index.html ตรง ๆ จากเครื่อง (file://) จะไม่สามารถติดตั้งเป็นแอปได้ เพราะ Chrome ต้องการ HTTPS
2. ตรวจสอบว่าอัปโหลดไฟล์ **ทุกไฟล์ในโฟลเดอร์** ขึ้น GitHub แล้ว (รวม manifest.json, sw.js, icons/)
3. ลองวิธีติดตั้งแบบ manual: เปิด Chrome บนมือถือ > กดจุดสามจุดมุมขวาบน > มองหา "ติดตั้งแอป" หรือ "เพิ่มไปยังหน้าจอโฮม"
4. ถ้าเคยเปิดเว็บนี้ค้างไว้ก่อนอัปโหลดไฟล์ครบ ให้ปิดแท็บแล้วเปิดใหม่ หรือล้างแคชเบราว์เซอร์ก่อน
5. บนมือถือ iPhone (Safari) ไม่มีปุ่ม "ติดตั้ง" แบบ Chrome ให้กด ปุ่มแชร์ (กล่องมีลูกศรขึ้น) > "เพิ่มไปที่หน้าจอโฮม"

## ตั้งค่าให้ "จุ้ย" คุยได้ทุกเรื่องด้วย Claude API

ค่าเริ่มต้น จุ้ยจะช่วยจดงาน/นัดหมาย/เงินแบบพื้นฐาน (ไม่ใช่ AI เต็มรูปแบบ)

ถ้าต้องการให้จุ้ยคุยได้ทุกเรื่องเหมือน AI จริง:

1. ไปที่ https://console.anthropic.com เพื่อขอ API key (เริ่มต้นด้วย `sk-ant-...`)
2. เปิดแอป MomFlow > กดไอคอนเฟือง (ตั้งค่า) มุมขวาบน
3. ใส่ API key ในช่อง "ผู้ช่วย AI — จุ้ย" แล้วกดติ๊กถูก
4. กลับไปที่แท็บ "จุ้ย" แล้วลองคุยได้ทุกเรื่องเลย

คีย์จะถูกเก็บไว้ในเบราว์เซอร์ของอุปกรณ์นั้นเท่านั้น (ไม่ถูกส่งไปเซิร์ฟเวอร์อื่นนอกจาก Anthropic) — **ข้อควรระวัง:** การเรียก API ตรงจากเบราว์เซอร์แบบนี้เหมาะกับการใช้งานส่วนตัว ไม่ควรแชร์ลิงก์แอปที่มีคีย์ฝังอยู่ให้คนอื่นใช้ เพราะคีย์จะถูกเก็บแยกในเบราว์เซอร์ของแต่ละคนอยู่แล้ว แต่ควรเก็บคีย์เป็นความลับเสมอ



### หน้าหลัก (Home)
- งานวันนี้ / นัดหมายถัดไป / เงินคงเหลือเดือนนี้
- ปุ่ม "เพิ่มแบบไว" — พิมพ์ประโยคเดียว ระบบสร้างงาน + นัดหมาย + รายจ่ายให้พร้อมกัน
- ไทม์ไลน์รวมทุกอย่างของวันนี้

### Tab งาน (To-Do)
- เพิ่ม/ติ๊กเสร็จ/ลบงาน
- กรองตามหมวด: 🏠 บ้าน, 👧 ลูกเล็ก, 👦 ลูกโต, 💰 การเงิน, 🏥 สุขภาพ, 🔴 เรื่องด่วน
- ระบบเดาวันที่และหมวดให้อัตโนมัติจากข้อความ

### Tab ปฏิทิน (Calendar)
- มุมมองรายวัน / รายสัปดาห์ / รายเดือน
- สีแยกตามหมวดหมู่
- พิมพ์ "ค่าเทอม 4,500 บาท 15 ต.ค." ระบบจะตั้งเตือนล่วงหน้าให้อัตโนมัติ 30 / 7 / 1 วันก่อนถึงกำหนด

### Tab เงิน (Money)
- สรุปรายรับ-รายจ่าย-คงเหลือ อัตโนมัติ
- เพิ่มรายการแบบแชท เช่น "+ เงินเดือน 18000" หรือ "- ค่าอาหาร 120"

### Tab จุ้ย (AI Chat)
- หน้าตาเหมือนแชท LINE
- "จุ้ย" คือผู้ช่วย AI เพศชาย ที่จะแทนตัวเองว่า "จุ้ย" เสมอ พูดสุภาพแบบผู้ชาย (ลงท้ายด้วย ครับ/นะครับ)
- ถ้าใส่ Anthropic API key ในตั้งค่า จุ้ยจะคุยได้ทุกเรื่องแบบ AI เต็มรูปแบบ (ไม่ใช่แค่เรื่องในแอป)
- ถามได้เช่น "พรุ่งนี้มีอะไรบ้าง", "ค่าใช้จ่ายเดือนนี้เท่าไร"
- พิมพ์อะไรก็ได้ที่มีวันที่/จำนวนเงิน จุ้ยจะช่วยจดงาน นัดหมาย และรายจ่ายให้อัตโนมัติ (ทำงานได้แม้ไม่ใส่ API key)

## โครงสร้างไฟล์
```
index.html      หน้าหลักของแอป
style.css       ดีไซน์ทั้งหมด (ธีมสีพิงค์ rose)
app.js          โค้ดทำงานของแอป (ข้อมูล, การคำนวณ, จุ้ย AI)
manifest.json   ตั้งค่าแอป สำหรับติดตั้งบนมือถือ
sw.js           Service Worker สำหรับเปิดใช้แบบออฟไลน์
icons/          ไอคอนแอป (ใช้รูปที่ส่งมา)
```

## หมายเหตุ
- ข้อมูลทั้งหมดเก็บในเบราว์เซอร์ของอุปกรณ์นั้น ๆ (ยังไม่ใช่ระบบ cloud sync ข้ามอุปกรณ์)
- ถ้าอยากเริ่มข้อมูลใหม่ ไปที่ปุ่มตั้งค่า (รูปเฟือง) มุมขวาบน > "รีเซ็ตข้อมูลตัวอย่าง"
